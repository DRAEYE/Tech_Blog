# Cheet Sheet

### 字体

示例：

```un
**这是加粗的文字**
*这是倾斜的文字*`
***这是斜体加粗的文字***
~~这是加删除线的文字~~
```

效果如下：

**这是加粗的文字**
*这是倾斜的文字*
***这是斜体加粗的文字***
~~这是加删除线的文字~~



### 分割线

三个或者三个以上的 - 或者 * 都可以，但显示无区别

示例：

```undefined
---
----
***
*****
```



### 图片

**调整图片大小**

``` markdown
<img src="链接" width="宽度" height="高度" alt="图片名称" align=center>
```
